# vue-study

## Project setup
```
npm install
```
For yarn users, run:
```
yarn install
```

### Compiles and hot-reloads for development
```
npm run serve
```
For yarn users, run:
```
yarn run serve
```

### Compiles and minifies for production
```
npm run build
```
For yarn users, run:
```
yarn run build
```

### Run your tests
```
npm run test
```
For yarn users, run:
```
yarn run test
```

### Lints and fixes files
```
npm run lint
```
For yarn users, run:
```
yarn run lint
```
